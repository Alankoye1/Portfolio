# React Portfolio Website

A modern, responsive portfolio website built with React.js featuring smooth animations, clean design, and mobile-first approach.

## 🚀 Features

- **Modern Design**: Clean and professional layout with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **Fast Performance**: Optimized for speed and performance
- **Smooth Scrolling**: Seamless navigation between sections
- **Contact Form**: Working contact form with validation
- **Projects Showcase**: Interactive project gallery with live demos
- **Skills Display**: Animated skill bars and technology showcases

## 🛠️ Technologies Used

- React 18
- CSS Modules
- HTML5 & CSS3
- JavaScript ES6+
- Responsive Design
- Modern CSS Grid & Flexbox

## 📦 Installation

### Prerequisites

Before running this project, make sure you have the following installed:
- [Node.js](https://nodejs.org/) (v14.0.0 or higher)
- npm (comes with Node.js)

### Setup Instructions

1. **Clone or download the project** to your local machine
2. **Navigate to the project directory**:
   ```bash
   cd ReactJs
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm start
   ```
5. **Open your browser** and visit `http://localhost:3000`

## 🚀 Building for Production

To create an optimized production build:

```bash
npm run build
```

The build folder will contain the optimized files ready for deployment.

## 🎨 Customization

### Personal Information

Update the following files with your information:

1. **Components**: Update personal details in:
   - `src/components/Hero/Hero.js` - Name and title
   - `src/components/About/About.js` - About information and stats
   - `src/components/Contact/Contact.js` - Contact details
   - `src/components/Footer/Footer.js` - Footer information

2. **Skills**: Modify the skills array in `src/components/Skills/Skills.js`

3. **Projects**: Update the projects array in `src/components/Projects/Projects.js` with your own projects

4. **Styling**: Customize colors and design in the respective CSS module files

### Images

Replace the placeholder images with your own:
- Add your profile picture to the `public` folder
- Update image paths in the components
- Add project screenshots to showcase your work
